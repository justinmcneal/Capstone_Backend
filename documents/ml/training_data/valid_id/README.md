# Valid ID - Training Data

## What Goes Here
Government-issued identification documents.

## Philippine ID Examples
- PhilSys National ID (Philippine Identification System)
- Driver's License (LTO)
- Passport (DFA)
- UMID (Unified Multi-Purpose ID)
- SSS ID
- PhilHealth ID
- Postal ID
- Voter's ID (COMELEC)
- PRC ID (Professional Regulation Commission)
- Senior Citizen ID
- PWD ID
- TIN ID

## Image Requirements
- **Format:** JPG, JPEG, or PNG
- **Minimum Size:** 224 x 224 pixels
- **Quality:** Clear, readable text
- **Content:** Front of ID, full document visible

## Target Count
**50-100 images minimum**

## Where to Find
1. **Kaggle:** https://www.kaggle.com/datasets/trainingdatapro/identity-document-image-dataset
2. **MIDV-500:** https://github.com/fcakyon/midv500
3. **Roboflow:** https://universe.roboflow.com/search?q=id+card
4. **Google Images:** Search "Philippine ID sample" (use for reference only)

## Variations to Include
- Different ID types (mix of all listed above)
- Different lighting conditions
- Slightly angled shots
- Various backgrounds
- Both old and new ID formats

## ⚠️ Privacy Note
- Use sample/template IDs when possible
- Blur or redact sensitive information on real IDs
- Do NOT commit real personal data to git
