# Selfie with ID - Training Data

## What Goes Here
Photos of a person holding their ID next to their face.

## Purpose
Anti-fraud verification - ensures the person uploading matches the ID.

## Image Requirements
- **Format:** JPG, JPEG, or PNG
- **Minimum Size:** 224 x 224 pixels
- **Quality:** Both face and ID clearly visible
- **Content:** Person's face + ID document in same frame

## Target Count
**30-50 images minimum**

## Characteristics to Look For
- Face clearly visible (not blurry)
- ID held near face level
- ID text readable (or at least document shape visible)
- Various backgrounds (home, office, outdoors)
- Different lighting conditions

## Where to Find
1. **Kaggle:** Search "selfie verification", "face id verification"
2. **Roboflow:** https://universe.roboflow.com/search?q=face+id
3. **Stock Photos:** Search "person holding id card"
4. **Generate:** Team members take sample selfies with printed ID templates

## How to Generate Samples
1. Print sample ID templates on paper
2. Have team members take selfies holding the paper
3. Vary lighting, angles, and backgrounds
4. This creates realistic training data

## Variations to Include
- Different people (age, gender)
- ID held on left/right side
- ID held at different angles
- Indoor/outdoor settings
- Good and slightly poor lighting

## ⚠️ Privacy Note
- Use fake/sample IDs for generated photos
- Get consent from anyone photographed
- Do NOT use real personal IDs
